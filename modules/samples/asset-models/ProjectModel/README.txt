Project Governance Model
========================

Project governance model can be use to govern differnet Projects. It allows to store data related to project such as Project name, Project manager, Project team members etc. Also this sample includes a sample report generator to generate reports about the currently existing projects.

This model includes the following modules,
1. Registry extension file.
   *project.rxt
2. Reporting template.
   *project_template.jrxml
3. Report Generator code.


Populate the model
==================
You can populate the model by executing ant run-project command inside $CARBON_HOME/samples/asset_models/Populator directory.
