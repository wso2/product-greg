Application Governance Model
============================

Application governance model can be use to govern different applications. It allows you to store data related to Applications such as Application name, Application version, Application description etc. Also this sample includes a sample report generator to generate reports about the currently existing applications.

This model includes the following modules,
1. Registry extension file.
   *application.rxt
2. Reporting template.
   *application_template.jrxml
3. Report Generator code.


Populate the model
==================
You can populate the model by executing ant run-application command inside $CARBON_HOME/samples/asset_models/Populator directory.
